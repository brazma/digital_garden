# Lexique US - FR 

|                                                            |                                                              |
| ---------------------------------------------------------- | ------------------------------------------------------------ |
| **Termes anglais**                                         | **Termes français**                                          |
| st : stitch                                                | m : mailles                                                  |

| slst : slip stitch                                         | mc : maille coulée                                           |
| sc : single crochet (US)  <br>dc : double crochet (UK)     | ms : [[maille serrée]]                                       |
| hdc : half double crochet (US)  <br>htr : half treble (UK) | db : [[demi bride]]                                          |
| dc : double crochet (US)  <br>tr : treble (UK)             | b : [[bride]]                                                |
| tr : treble  (US)  <br>dtr : double treble (UK)            | Db : [[double bride]]                                        |
| dtr : double treble (US)  <br>trip tr : triple treble (UK) | Tb : triple bride                                            |
| Inc : increase                                             | Aug : augmentation                                           |
| Inv-dec : invisible decrease                               | Dim-inv : diminution invisible                               |
| Dec : decrease                                             | Dim : diminution                                             |
| Sk : skip                                                  | Passer                                                       |
| Yo : yarn over                                             | Faire un jeté                                                |
| … blo : … back loop only                                   | Dans le brin arrière seulement                               |
| … flo : … front loop only                                  | Dans le brin avant seulement                                 |
